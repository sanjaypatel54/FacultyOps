Project Overview

FacultyOps is a web-based faculty profile management platform designed for presenting educational qualifications, designation, department, research areas, specialisation, contact information and detailed faculty profile summaries in a professional format.


DevOps Scope

  • Version control using Git and GitHub.
  • Code editing using VS Code.
  • Static hosting through AWS S3.
  • Automated CI/CD pipeline for deployment.
